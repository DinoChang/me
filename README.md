# Dino's Portfolio Terminal

> 一個互動式終端機風格的個人頁面

## Demo

**[https://dinochang.github.io/me/](https://dinochang.github.io/me/)**

## 特色

- 終端機風格介面，模擬真實 bash 操作體驗
- 支援多種指令互動，包含彩蛋與隱藏功能
- Matrix 背景動畫效果
- 響應式設計，支援手機與桌面瀏覽
- 純前端實作，無需後端服務

## 可用指令

| 指令 | 說明 |
|------|------|
| `help` | 顯示所有可用指令 |
| `whoami` | 顯示基本資訊 |
| `skills` | 查看技能樹 |
| `experience` | 查看專案經歷 |
| `contact` | 聯絡方式 |
| `neofetch` | 系統資訊風格展示 |
| `clear` | 清除畫面 |

還有更多隱藏指令等你發掘！

## 技術棧

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Fira Code, Noto Sans TC)
- Canvas API (Matrix 動畫效果)
